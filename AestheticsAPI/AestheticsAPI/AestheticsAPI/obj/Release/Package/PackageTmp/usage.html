Slack - Custom Slash Commands

 

To use the AestheticAPI in Slack as a custom slash command, follow the steps below:

 

1. Create a new Slash Command with the following settings:

 

Command: /aesthetic

URL: aestheticAPI.gucc.io/api/Fullwidth

Method: POST

Token: (your token)

Name: AestheticBot

Description: Converts inputted text into ａｅｓｔｈｅｔｉｃ.

Usage Hint: [message to turn into aesthetic]

Escape: Off

Label: Converts inputted text into ａｅｓｔｈｅｔｉｃ, full-width text.

 

2. Save your  custom integration, and test your command within Slack.
